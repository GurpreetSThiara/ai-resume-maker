# Keyboard Shortcuts for Resume Editor

This document outlines the available keyboard shortcuts to enhance your productivity while using the resume editor.

## General Shortcuts

- **Save Resume**: `Ctrl + S`
  - **Action**: Saves the current state of your resume to local storage, allowing you to continue where you left off without losing any changes.

## Navigation

- **Next Step**: `Ctrl + ArrowRight`
  - **Action**: Moves to the next section in the resume creation process, making it easier to navigate without using the mouse.

- **Previous Step**: `Ctrl + ArrowLeft`
  - **Action**: Returns to the previous section, allowing for quick corrections and edits.
